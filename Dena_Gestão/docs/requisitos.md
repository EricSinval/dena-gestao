# Requisitos — Dena Gestão

## Objetivo

Desenvolver um sistema de gestão para a Dena Moda Fashion que permita centralizar e controlar informações relacionadas a produtos, estoque, clientes, pedidos, produção e movimentações financeiras.

## Usuários

Inicialmente, o sistema será utilizado pela responsável administrativa da empresa.

## Funcionalidades do MVP

- Login de usuário;
- Cadastro de categorias;
- Cadastro de produtos;
- Consulta de produtos;
- Edição de produtos;
- Inativação de produtos;
- Busca de produtos;
- Filtro por categoria;
- Filtro por status;
- Controle básico de estoque;
- Identificação de produtos com estoque abaixo do mínimo.

## Tecnologias

- HTML;
- CSS;
- JavaScript;
- Python;
- Django;
- PostgreSQL;
- Power BI;
- Git e GitHub.
